A minimal terminal-based to-do list with 10 tasks. You can mark tasks as completed or incomplete.  

How to Run  
	1.	Make sure you have Python 3 installed.  
	2.	Run the script in the terminal:  
      python3 list.py  
   
How to Use  
	•	The list of 10 tasks is displayed with [ ] for incomplete and [x] for completed.   
	•	Enter the task number to toggle its status.  
	•	Enter q to quit the program.
